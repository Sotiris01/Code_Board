# C++ Content Directory

This directory contains C++ educational content for Code Board.

## Structure (Future)

```
content/cpp/
├── exercises/         # C++ programming exercises
├── templates/         # Code templates
└── examples/          # Example programs
```

## File Extension

C++ files use the `.cpp` extension.

## Adding Content

To add new C++ content:
1. Create appropriate subdirectories
2. Add `.cpp` files with your content
3. Content will automatically appear in the file browser when C++ language is selected
